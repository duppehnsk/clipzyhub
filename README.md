🎬 Clipzy
Clipzy — это инструмент и бот для скачивания и работы с короткими видео (Reels, Shorts и др.), созданный для удобного и быстрого получения контента.
🚀 Возможности
  📥 Скачивание Instagram Reels
  🎥 Поддержка YouTube Shorts
  📝 Получение описания видео
  ⚡ Быстрая обработка ссылок
  🤖 Удобное использование через Telegram-бота
  🛠️ Технологии
Python + Telegram Bot API
